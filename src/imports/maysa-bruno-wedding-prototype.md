Crie um protótipo completo de site de casamento moderno, elegante e minimalista para "Maysa & Bruno".

Objetivo do site:
Permitir que convidados enviem fotos tiradas durante o evento, salvando essas fotos para o casal. O foco principal do projeto é a galeria colaborativa de fotos.

Estilo visual:
- Romântico, elegante e clean
- Paleta de cores: branco, bege/areia e detalhes dourados suaves
- Tipografia: título com fonte serifada elegante (estilo Playfair ou Cormorant) e textos com sans-serif leve e moderna
- Layout sofisticado, com bastante respiro, sombras suaves e bordas arredondadas
- Animações suaves de fade-in e scroll

Crie as seguintes telas:

1) HOME
- Hero section com foto grande do casal
- Nome "Maysa & Bruno" centralizado
- Data do casamento
- Local do evento
- Contagem regressiva
- Texto curto de agradecimento aos convidados
- Botão principal destacado: "Enviar suas fotos"
- Botão secundário: "Ver galeria"
- Espaço para QR Code
- Rodapé simples com frase romântica

2) TELA DE ENVIO DE FOTOS
- Layout otimizado para mobile
- Área grande de upload com drag-and-drop
- Botão "Selecionar fotos"
- Permitir múltiplas fotos
- Campo opcional: Nome do convidado
- Campo opcional: Mensagem para os noivos
- Checkbox de autorização de uso das imagens
- Barra de progresso de upload
- Mensagem de confirmação elegante após envio

3) GALERIA
- Grid estilo Pinterest
- Cards com bordas arredondadas
- Nome do convidado exibido discretamente
- Filtro no topo: Mais recentes / Destaques
- Modal ao clicar na imagem (visualização ampliada)
- Botão discreto de compartilhar
- Layout fluido e responsivo

4) TELA ADMIN (apenas wireframe visual)
- Lista de fotos enviadas
- Status: Aprovada / Pendente
- Botões: Aprovar / Rejeitar
- Botão: Baixar todas as fotos
- Layout simples e funcional

Requisitos de UX:
- Design mobile-first
- Interface extremamente simples para facilitar uso por convidados de todas as idades
- Botões grandes e claros
- Hierarquia visual bem definida
- Fluxo intuitivo em 2 cliques para envio de fotos

Crie o protótipo com navegação simulada entre as telas.
